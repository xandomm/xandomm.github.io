# xandomm.github.io

Personal portfolio site for Alexandre Machado — Full-Stack Software Engineer (React, React Native, Python).

Static site (HTML/CSS/JS, no build step) served via GitHub Pages. Bilingual (EN/PT) with a language toggle in the nav.

## Structure

- `index.html` — page content
- `assets/css/style.css` — styles
- `assets/js/main.js` — language toggle, mobile nav, scroll-reveal animations
- `assets/Alexandre_Machado_Resume.pdf` — downloadable résumé

## Local preview

```
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
